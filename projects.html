Junior Unity Developer - Dominic Harris Studios.

"Unseen"

Unseen is an interactive portrait of undiscovered butterflies that have never been witnessed or documented by humans. Using cutting-edge AI-generation techniques to train and create 13,500+ Butterflies on sample data provided by natural history museums across the world. Currently on show at the Halcyon Gallery located in New Bond Street, London.

More information can be found here

•Applied AI and UE4 (Blueprints) to create real-time collages with 13,500 butterflies using advanced colour theory techniques and python backend which is responsible for displaying triadic colour palletes and colour matching system in place.

"Metamorphosis Unseen"(Nvidia Jetson TX2)

Metamorphosis Unseen is practically using the same backend system as the larger Unseen format however displaying on the Nvidia Jetson TX2 platform with some adjustments to ensure stable performance on the weaker platform.

• The use of Unreals' Vulkan rendering pipeline to ensure more control over GPU and lower level CPU usage for the Jetson TX2.

• Ensured early R&D for the Jetson TX2 platform to assess whether or not haivng the same backend and base system would be able to withstand a stable framerate
